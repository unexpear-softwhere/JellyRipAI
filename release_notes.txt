JellyRip AI v1.0.21 — audit-driven cleanup.  Default Gemini model

### What's New in 1.0.21

Provider stack
- Default Gemini model bumped from gemini-2.0-flash to
  gemini-2.5-flash.  Matches the price/perf tier shift we did for
  Claude in v1.0.20.  Older models stay available in the dropdown.
- credential_store.py docstring example updated to match.

Documentation
- README "Main Workflows" added "(some testing)" / "(not tested)"
  qualifiers to match MAIN's honesty about test status.

Bundle / repo
- Removed gui_qt/qss/warm.qss (empty 0-byte placeholder).
- pyproject.toml keywords dropped tkinter, added pyside6/qt/claude/ai.
- .gitignore additions: dashboard.html (defensive), *.tmp.

### What's NOT in this release

No engine or workflow changes.  Chat sidebar, AI provider dialog,
diagnostics routing, on-device fallback all behave identically to
ai-v1.0.20.  Existing users with a saved Gemini configuration are
unaffected — their saved preference takes precedence over the new
default.
